# 뒤집기

[link](https://www.acmicpc.net/problem/1439)

## 내 풀이

- 방식 : 그리디
- 0에서 1, 1에서 0으로 변경될 때 마다 1씩 증가시킴
- 그 수를 2로 나눈 몫을 결과값으로 출력


**📰code**
```python
n = input()
t = '0 or 1'
answer = 0
for i in n:
    if i != t:
        answer +=1
        t=i
print(answer//2)
```
